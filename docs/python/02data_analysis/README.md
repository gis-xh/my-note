## 环境配置

&emsp;&emsp;本篇的内容，均是在虚拟环境 `data01` 中进行的。

> 创建虚拟环境

```sh
conda create -n data01 python=3.9
```

> 激活虚拟环境

```sh
conda activate data01
```

> 安装 JupyterLab

```sh
conda install -c conda-forge jupyterlab -y
```
